How to Draw a 'Basic Banana on a Table' using HTML and CSS.



  1. First make an ellipse and color it yellow.


    2. .BananaSkin{

    background-color: yellow;

    height: 100px;

    width: 100px;

    border-radius: 60%;

    }

  2. Then make a brown, rounded square under the yellow ellipse.


    3. .StalkRoundedSquare{

    background-color: brown;

    height: 90px;

    width: 80px;

    border-radius: 10;

    position:relative;

    margin-top:-85px;

    margin-left: 30px;

    z-index:-1;

    }

  3. Cover part of your picture with a gray, rounded square to make your banana appear! Add a box-shadow to the rounded, gray square to cast a shadow on the banana shape.


    4. .ShadowRoundedSquare{

    background-color: gray;

    height: 100px;

    width: 100px;

    border-radius: 40px;

    margin-top: 25px;

    margin-left: 40px;

    position: absolute;

    box-shadow: 0 -3px 20px 0px gray;

    }

  4. Finally, make a gray, rounded table to sit the banana shape on. Give the table a box-shadow to make it pop out.


    5. .Table{

    background-color: gray;

    height: 200px

    width: 200px;

    border-radius: 10%;

    position:relative;

    margin-top:-100px;

    margin-left: 15px;

    z-index:-2;

    box-shadow: 0 50px 50px 5px ;

    }
